# UBencode 助手

UBencode 助手是 MoviePilot V2 插件，用于绑定 UBencode 压制中心账号、检测 MoviePilot 中的 UBits 站点信息、领取任务并把源种推送到已配置下载器。

## 1.1.0 功能

- 使用验证码绑定压制中心账号，密码不进入插件配置。
- 自动检测 UBits Cookie，并从 RSS 或页面链接提取 Passkey。
- 选择 MoviePilot 已启用的 qBittorrent 或 Transmission 下载器。
- 查看待测压、待正压、已领取和已完成任务。
- 手动领取任务并下载源种，使用种子 info hash 去重。
- 按 CRON 拉取 UBencode 客户端事件，使用服务端游标和事件 ID 双重去重。
- 支持快速测压、精准测压的每轮结果、完成和失败通知。
- 支持正压进度节点、完成、失败和测压/正压队列摘要通知。
- 最近 80 条事件保存在插件数据区，详情页展示最近 12 条。
- 配置页按账号、站点下载器和通知类别分区，并自动适配 MoviePilot 浅色/暗色主题。

客户端事件上报失败时会写入独立本地队列，网络恢复或下次登录后继续上传，不阻塞测压和正压流程。
